Quarter Project: First Person Ninja Game

by Davin Johnson

For my quarter project I made a playable video game. It is a first person perspective game that uses animated skinned meshes, model deformation, and billboard animation for its graphics.

The Game

The game is first person and is controlled using mouse and keyboard. The goal of the game is to collect all the crates located around the map while avoiding the Ninja enemies scattered around. If they detect you they will run towards you until you either defeat them or get too far away from them.

Controls

Movement is controlled by WASD keys, and look is controlled by moving the mouse. LCTRL can also be used to crouch, which will make enemy Ninjas detect you in a smaller radius. Also, if you manage to crouch walk close enough to a ninja you can defeat them in one sword swing!

There is a sword swing attack and a throwing star attack that can be used to defeat the Ninjas and which are controlled with Mouse Click and E respectively.

Skinned Mesh Animation

Enemy Ninjas

The game uses skinned mesh animations for the enemy Ninjas and for the player attack animations.

The enemy Ninjas have three animations; idle, walking, and running. When the player is not near a Ninja it will alternate between standing idle and walking in a random direction.

Player Attacks

The player has two attacks they can use and each has its own animation.

Deformation

Deformation is used to add a swaying effect to the trees in the map simulating a breeze to add some life to the game.

Billboard Animation

Finally, a puff of smoke appears whenever a player defeats an enemy Ninja and these are rendered using billboard animation
